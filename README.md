# TypeScript Course

A beginner-friendly TypeScript course with practical exercises.

## Setup

1. **Install Node.js** (if not already installed)
   - Download from https://nodejs.org/

2. **Install dependencies**
   ```bash
   npm init -y
   npm install typescript ts-node @types/node --save-dev
   ```

3. **Initialize TypeScript config**
   ```bash
   npx tsc --init
   ```

## Running Lessons

Run any lesson file with:
```bash
npx ts-node 01-introduction.ts
```

## Course Outline

| Lesson | Topic | Concepts |
|--------|-------|----------|
| 01 | Introduction | Types, variables, type inference |
| 02 | Functions | Parameters, return types, arrow functions |
| 03 | Arrays & Loops | Typed arrays, for/forEach/map/filter |
| 04 | Objects & Interfaces | Interfaces, optional properties, extending |
| 05 | Classes | Constructors, access modifiers, inheritance |
| 06 | Unions & Guards | Union types, literal types, type narrowing |
| 07 | Generics | Generic functions, classes, constraints |
| 08 | Bonus Challenge | Build a complete Task Manager |

## Tips

- Complete the exercises at the end of each lesson
- Uncomment the error examples to see TypeScript catch mistakes
- Use VS Code for the best TypeScript experience (autocomplete, error highlighting)

## Next Steps

After completing this course:
- Learn about decorators
- Explore utility types (Partial, Pick, Omit, etc.)
- Try building a project with React + TypeScript or Node.js + TypeScript
